# Proyecto_SP_7_cars
Proyecto sprint 7 tripleten sobre carros enfocado al uso de git y de streamlit 
Este proyecto se hizo siguiendo los siguientes pasos
1. Se crea el repositorio en Github con su archivo README.md e ignore correspondiente
2. Se clona de manera loca y por medio del uso de la terminal se crean las carpetas y archivos base
3. Se crea un notebook de jupyter para hacer pruebas casilla a casilla de cada una de las funciones con base en el csv obtenido
4. Se unen todas estas funciones de una manera organizada y documentada en el archvio app.py
5. Se publica a traves de render obteniendo el siguiente enlace https://proyecto-sp-7-cars-1.onrender.com/
